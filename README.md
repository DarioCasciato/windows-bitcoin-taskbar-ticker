# Windows Bitcoin Taskbar Ticker

**Windows Bitcoin Taskbar Ticker** is a lightweight application that displays the current Bitcoin price in USD directly on your Windows taskbar. Stay updated with real-time Bitcoin prices without having to open any webpage.

### Setup

1. **Run the Executable:**

   Double-click `windows-bitcoin-taskbar-ticker.exe` to launch the application. It will appear in your system taskbar, displaying the current Bitcoin price.

2. **Add to Windows Autostart:**

   To ensure the application starts automatically with Windows, follow one of the methods below:

3. **Open Startup Folder:**

    - Press `Win + R`, type `shell:startup`, and press **Enter**. This opens the Startup folder for the current user.

4. **Create Shortcut:**

    - Right-click inside the Startup folder, navigate to **New > Shortcut**.
    - Browse to the location of `windows-bitcoin-taskbar-ticker.exe`, select it, and click **Next**.
    - Name the shortcut (e.g., `Bitcoin Taskbar Ticker`) and click **Finish**.

The application will now launch automatically every time you log into Windows.

## Usage

- **View Bitcoin Price:**

  Hover over the Bitcoin icon to see the latest Bitcoin price in USD.

- **Exit Application:**

  Right-click the Bitcoin icon and select **Exit** to close the application.
